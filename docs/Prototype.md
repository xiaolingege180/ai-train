```
主色调：
- Primary: #4F46E5 (Indigo 600)

- Primary Dark: #3730A3 (Indigo 800)

辅助色：
- Success: #10B981 (Emerald 500)
- Warning: #F59E0B (Amber 500)

- Info: #3B82F6 (Blue 500)

- Background: #F9FAFB (Gray 50)
- Surface: #FFFFFF
- Text Primary: #111827 (Gray 900)
- Text Secondary: #6B7280 (Gray 500)
- Border: #E5E7EB (Gray 200)
```

```
中文字体：
- 标题：PingFang SC, Microsoft YaHei
- 正文：PingFang SC, -apple-system

英文字体：
- 标题：Inter, system-ui
- 正文：Inter, -apple-system

- H1: 32px / 40px line-height / 700 weight
- H2: 24px / 32px / 600
- H3: 20px / 28px / 600
- H4: 18px / 26px / 500
```

## 3. 响应式适配

### 3.1 断点设置
```
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
```

### 3.2 移动端适配要点
- 导航栏变为汉堡菜单
- 课程列表单列显示
- 侧边栏变为底部 Tab 或折叠菜单
- 视频播放器全宽显示
- 操作按钮底部固定

## 4. 交互动效规范

### 4.1 过渡动画
```
- 默认过渡: transition-all duration-300 ease-in-out
- 按钮悬停: scale(1.02) + shadow
- 卡片悬停: translateY(-4px) + shadow-lg
- 页面切换: fade + slide
- 加载状态: skeleton pulse
```

### 4.2 反馈动效
```
- 成功: checkmark scale + green flash
- 错误: shake + red border
- 加载: spinner rotate
- 新消息: badge bounce
```
